Elementos Semanticos

<article>

Representa um conteúdo independente, como uma postagem de blog ou artigo, que pode ser entendido por si só.

Exemplo: https://www.w3schools.com/TAGS/tag_article.asp

<aside>

Define a área de cabeçalho de uma página ou seção, geralmente contendo títulos, logotipos e menus de navegação.

Exemplo: https://www.w3schools.com/TAGS/tag_aside.asp

<details>

O elemento HTML details é usado como uma ferramenta de onde o usuário irá obter informações adicionais.

Exemplo: https://www.w3schools.com/TAGS/tag_details.asp

<figcaption>

O elemento HTML Figcaption representa uma legenda ou uma legenda associada com uma figura ou ilustração descrita pelo resto dos dados do elemento que seu elemento pai.

Exemplo: https://www.w3schools.com/TAGS/tag_figcaption.asp

<figure>

O elemento HTML figure representa conteúdo autocontido, potencialmente com uma legenda opcional, que é especificada usando o elemento. A figura, sua legenda e seu conteúdo são referenciados como uma única unidade.

Exemplo: https://www.w3schools.com/TAGS/tag_figure.asp

<footer>

O elemento HTML footer de Rodapé representa um rodapé para o seu sectioning content (conteúdo de seção) mais próximo ou sectioning root elemento (ou seja, seu parente mais próximo)

Exemplo: https://www.w3schools.com/TAGS/tag_footer.asp

<header>

O elemento HTML header representa um grupo de suporte introdutório ou navegacional. Pode conter alguns elementos de cabeçalho mas também outros elementos como um logo, seções de cabeçalho, formulário de pesquisa, e outros.

Exemplo: https://www.w3schools.com/TAGS/tag_header.asp

<main>

O elemento main define o conteúdo principal dentro do body em seu documento ou aplicação. Entende-se como conteúdo principal aquele relacionado diretamente com o tópico central da página ou com a funcionalidade central da aplicação.

Exemplo: https://www.w3schools.com/TAGS/tag_main.asp

<mark>

O Elemento HTML mark representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.

Exemplo: https://www.w3schools.com/TAGS/tag_mark.asp

<nav>

O Elemento HTML nav de Navegação representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.

Exemplo: https://www.w3schools.com/TAGS/tag_nav.asp

<section>

O elemento HTML section representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.

Exemplo: https://www.w3schools.com/TAGS/tag_section.asp

<summary>

O elemento HTML summary é utilizado como um sumário ou legenda para o conteúdo de um elemento

Exemplo: https://www.w3schools.com/TAGS/tag_summary.asp

<time>

O elemento HTML time representa o tempo tanto no formato de 24 horas ou como uma data precisa no calendário Gregoriano (com informações opcionais de tempo e fuso horário)

Exemplo: https://www.w3schools.com/TAGS/tag_time.asp
